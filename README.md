## Overview
A lightweight, interactive bot built with Streamlit that helps users calculate and visualize their ideal term insurance coverage. 

## The Problem
Calculating term insurance can be confusing and overwhelming for the average person. I wanted to build a straightforward tool that simplifies the math and provides immediate, actionable estimates without requiring a spreadsheet.

## Features
- **Interactive UI:** Built entirely in Streamlit for a clean, user-friendly experience.
- **Dynamic Calculations:** Instantly updates coverage estimates based on user inputs (age, income, liabilities).

## Tech Stack
- **Frontend/Framework:** Streamlit
- **Language:** Python
- **Automation:** Make.com
